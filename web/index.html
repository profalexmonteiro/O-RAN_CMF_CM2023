Parâmetros das Estações Base

Usuários

User Profiles

Cada perfil define o bitrate exigido pelo usuário e a probabilidade de atribuição aleatória desse perfil.

Traffic Generation

Cada usuário gera tráfego por tentativas de conexão. A conexão só é estabelecida quando alguma estação base possui PRBs livres suficientes, e pode sofrer handover após evento A3.

Propagation Model

O modelo UMa (Urban Macro) do 3GPP TR 38.901 é utilizado para calcular pathloss e margens adicionais de propagação.

RIC Configuration

Todas as estações base são gerenciadas por um único near-RT RIC. Os xApps MRO e MLB são executados simultaneamente para ajustar handover e balanceamento de carga.
